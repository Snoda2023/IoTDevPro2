# IoTDevPro2
## 役割分担
・ディスプレイをやる人
・MQTT通信をやる人
・ジョイコンとCdSをやる人
・I2Cセンサとそれ以外のデバイスをやる人
・全員の成果物を管理して結合する人

## 構成
MCP3208に接続するデバイス: ジョイコンx2、CdS
I2C接続するセンサ: BME280 or ジェスチャセンサ
それ以外のデバイス: サーボモーターx1、超音波センサx1
I2C接続するディスプレイ: RGBディスプレイx1
